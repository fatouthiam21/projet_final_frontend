DOSSIER IMAGES - Urban Style
============================

Place tes images dans les sous-dossiers correspondants.
Depuis le code, elles seront accessibles via : /images/dossier/fichier.jpg

Structure :
-----------
images/
  jeans/          → photos de jeans (jean-slim.jpg, jean-baggy.jpg ...)
  cargos/         → photos de cargos
  t-shirts/       → t-shirts, hoodies, polos
  chemises/       → chemises
  sneakers/       → sneakers et baskets
  chaussures/     → derby, mocassins, boots
  hero/           → grandes images pour le slider d'accueil (1920x1080 recommandé)
  categories/     → vignettes de catégories (800x600 recommandé)
  banners/        → bannières promotionnelles (1200x400 recommandé)

Formats acceptés : .jpg, .jpeg, .png, .webp
Taille recommandée : max 500ko par image (utilise https://squoosh.app pour compresser)

Exemple d'utilisation dans le seed.js :
  url: '/images/jeans/mon-jean.jpg'
